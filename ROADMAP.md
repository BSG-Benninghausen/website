# Roadmap

Der Backlog dieses Produkts wird **zentral im Hauptrepo** geführt — als **GitHub Issues + das
Project »Vereins-Baukasten Roadmap«** (Owner `crypticalcode`). Forks haben Issues deaktiviert; offene
Vorhaben stehen daher upstream:

- Index & Gruppierung: [crypticalcode/vereins-baukasten · ROADMAP.md](https://github.com/crypticalcode/vereins-baukasten/blob/main/ROADMAP.md)
- Issues: <https://github.com/crypticalcode/vereins-baukasten/issues>
- Board: GitHub-Project »Vereins-Baukasten Roadmap«

Fork-spezifische Arbeit (Branding, club-eigene Seiten/Seeds) läuft als **PR in diesem Fork**;
vereins-neutrale Beiträge gehen via Mergeback nach upstream (siehe
[`CONTRIBUTING.md`](CONTRIBUTING.md)).
